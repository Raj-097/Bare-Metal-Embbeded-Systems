# Statistical Analysis of an Array

## Author

Rajkumar Saravanakumar

## Overview

This project implements basic statistical operations on an array of unsigned 8-bit integers in C. The program sorts the data set and computes the mean, median, maximum, and minimum values. Functions are also provided to display the array contents and the calculated statistics.

## Features

* Array sorting (descending order)
* Mean calculation
* Median calculation
* Maximum value identification
* Minimum value identification
* Array and statistics display
